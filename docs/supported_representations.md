# Supported Allele Representations

## DNA Sequence allele
* HGVS Expression (including reference sequence identifier)
* VCF
* ClinGen Allele
* GA4GH Variant

## Protein allele
* HGVS Expression (including HGNC gene identifier)
* ClinGen Allele
* GA4GH Variant

## Copy number variant
* ISCN nomenclature
* VCF
* ClinGen Allele v0.2 (future)
* GA4GH Variant
